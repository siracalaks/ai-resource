# Vektra — dashboard

The zero-install dashboard lives here.

- **Open `dashboard.html`** (double-click) → the Library tab works offline immediately.
- To light up the live tabs (Intel / Ask), enter your Supabase + webhook URLs in the **Settings** tab.

Full docs: **[../README.md](../README.md)** (English) · **[../README.tr.md](../README.tr.md)** (Türkçe) · setup: **[../docs/INSTALL.md](../docs/INSTALL.md)**

`harvest.mjs` is an optional standalone Node script (HN + GitHub, zero deps) for a quick local
harvest without n8n.
